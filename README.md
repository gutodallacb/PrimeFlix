# Prime Flix

Prime Flix é uma aplicação web desenvolvida em **React** que permite ao usuário explorar filmes em exibição, ver detalhes, salvar favoritos e assistir trailers.  
As informações são obtidas através da **API do The Movie Database (TMDb)**.

---

## Demo: 
https://primeflix-gutodallacb.vercel.app/

---

## Tecnologias Utilizadas

- **React**
- **Vite**
- **React Router DOM**
- **Axios**
- **React Toastify**
- **TMDb API**
- **HTML5, CSS3 e JavaScript**

---

## Funcionalidades

- **Listar filmes em destaque**
- **Visualizar detalhes de um filme específico**
- **Salvar filmes na lista de favoritos**
- **Remover filmes da lista de favoritos**
- **Pesquisar trailers no YouTube**
- **Interface responsiva e amigável**

---
